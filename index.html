INFORMACION GENERAL

Bienvenidos al OVI de diseños de sitios WEB (curso 301122) en el cual se plasma lo realizado en actividades previas como es la del desarrollo, creación del Guion y maquetación para un OVI y la actividad correspondiente a la fase 3, que es la Construcción: Crear un OVI en formato WEB con HTML5 y CSS3.


OBJETIVOS DEL OVI

  • Plasmar y poner en práctica lo visto en las actividades previas, como es la creación de un repositorio en GITHUB y el uso de lenguajes de programación.
  • Demostrar el manejo de las herramientas como, GIT, CCS3 y por supuesto el uso de HTML5 y de más aprendido durante el desarrollo de este curso.
  • Desarrollar lo propuesto en la rúbrica de la actividad del curso diseño de sitios web.
  • Aprender a programar de manera básica, sitios web y publicarlos en internet a través de un repositorio, en este caso GITUB-PAGE para que este se pueda visualizar.


OJBETIVOS DEL CURSO

El curso de diseño de sitios web de la UNAD, tiene como objetivos el de lograr que nosotros como estudiantes planeemos, diseñemos y realicemos sitios web, a través de lenguaje de programación HTML versión número 5, y CCS versión número 3, así mismo que trabajemos de manera colaborativa con nuestros compañeros en el desarrollo de estos sitios webs, con la ayuda de repositorios en este caso, GITHU y su posterior publicación, todo con el fin de que en un futuro lo podamos poner en practica en nuestra vida en caso de necesitarlo.


TEMATICAS REFERENCIA

Para el desarrollo de esta actividad se necesitó usar herramientas y conceptos que facilitaran la realización de la misma, dichos elementos son las siguientes:

  • HTML 5, Significa HyperText Markup Language, versión 5, ¿qué es esto? Un lenguaje en el que se escriben las páginas Web, que permite hacer enlaces (hipertexto) y que es leído por los navegadores para poder mostrar la información en la pantalla de tu ordenador de una forma correcta.
  • CSS, es un estándar diseñado principalmente para separar los atributos de presentación (como el color de fondo, el tamaño de la fuente o la sangría) del contenido. Aunque éste puede trabajar con muchos tipos de documentos, es mayormente utilizado como una forma de proveer información para presentarla en páginas web.
  • Github, es una plataforma creada para facilitar el desarrollo colaborativo de software, nos permite alojar proyectos en la web gratuitamente, por lo general de forma pública, aunque podemos alojar los proyectos de modo privado, si pagamos una pequeña suscripción mensual.
  • Visual Studio Code, es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código
